## 🚀 Getting Started

## Prerequisites
- Node.js (v18+)
  
---

### 1. Clone repository
```bash
git clone https://github.com/Shigosag/Shigosag-Dashboard.git
cd Shigosag-Dashboard
```

---

## ⚙️ Server Setup

```bash
cd server
npm install
npm run dev
```

Backend runs at: http://localhost:5000

## 🖥️ Client Setup

```bash
cd client
npm install
npm run dev
```

Frontend runs at: http://localhost:5173
